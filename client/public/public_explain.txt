/public
  index.html      # React 앱의 HTML 템플릿
  /assets         # 앱에서 사용하는 이미지, 아이콘, 로고 등
    logo.png
    favicon.ico
  manifest.json   # PWA 관련 설정 (Progressive Web App)
  robots.txt      # 검색 엔진 크롤러 설정


index.html:
React 앱의 최초 HTML 템플릿입니다. 기본적인 구조와 루트 div 요소가 포함되어 있으며, React가 JavaScript를 통해 DOM을 동적으로 렌더링합니다.


manifest.json (PWA 관련):
Progressive Web App (PWA)를 구현하려면 이 파일에서 앱의 이름, 아이콘, 색상 등의 설정을 지정합니다.


정적 자원 (이미지, 아이콘 등):
public/assets 폴더에는 앱에서 사용하는 정적 자원들이 들어갑니다. 예를 들어, 로고나 아이콘 파일이 포함됩니다.
React에서는 public 폴더 내의 자원을 사용할 때는 public/을 경로에 포함하지 않고, 예를 들어 /assets/logo.png처럼 사용할 수 있습니다.


robots.txt:
이 파일은 검색 엔진이 사이트를 크롤링하는 방법을 지정할 수 있습니다. 예를 들어, 사이트의 특정 페이지를 크롤링하지 않도록 지정할 수 있습니다.
